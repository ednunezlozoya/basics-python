------------------------------------------POST--------------------------------------------
http://laboratoriosacme.com/citas

[POST] 	Registra o agenda una cita para algun paciente para algun estudio determinado
Accept:	application/json

Body:
{
   "paciente-id":1,
   "estudio-id",3,
   "fecha-cita":"10-05-2023",
   "hora-cita":"07:00"
}

Status Code: 	201 (Created)
Content-type:	application/json

{
   "cita-id",1001,
   "paciente-id":1,
   "estudio-id",3,
   "fecha-cita":"10-05-2023",
   "hora-cita":"07:00"
}
